# 💬 Customer Support AI Assistant - React UI

A modern **React + TypeScript** frontend for an AI-powered customer support system.  
This UI allows agents to interact with AI-generated response drafts, review them, and take actions like **edit, approve, or reject** before sending responses to customers.

---

## 🚀 Features

- 🧠 AI-generated response preview (from backend API)
- 💬 Chat-style customer query interface
- ✏️ Edit AI-generated drafts before approval
- ✅ Approve responses for final submission
- ❌ Reject responses with feedback logging
- 📜 Conversation history tracking
- 🔄 Real-time status indicators (Pending / Approved / Rejected)
- 📱 Responsive UI for desktop and tablet
